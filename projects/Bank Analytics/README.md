🏦 Digital Banking Intelligence Dashboard

An interactive Banking Analytics Dashboard built using **Power BI** to analyze customer, account, loan, and transaction data. This project focuses on transforming raw banking data into meaningful business insights through data modeling, DAX, and interactive visualizations.



📌 Project Overview

The objective of this project is to analyze banking operations and provide a clear overview of customer behavior, loan performance, account distribution, and transaction trends. The dashboard is designed to help business users make data-driven decisions using interactive reports.


🎯 Objectives

- Analyze customer and banking performance.
- Monitor loan portfolio and transaction trends.
- Understand customer distribution across cities.
- Evaluate account types and credit score distribution.
- Build an interactive dashboard for business decision-making.

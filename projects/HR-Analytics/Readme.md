📌 Project Overview

This project analyzes HR employee data to identify workforce trends, employee attrition, salary distribution, and performance insights using SQL, Python, and Power BI.

🛠 Tools Used
Python (Pandas)
SQL (MySQL)
Power BI


📈 Dashboard KPIs
Total Employees
Active Employees
Attrition Rate
Attrition Count
Average Salary
Average Age

📊 Dashboard Features
Department-wise Employee Distribution
Attrition by Department
Performance Rating Analysis
Age Group Distribution
Department-wise Average Salary
Job Satisfaction Matrix


🔍 Key Insights
Total Employees: 4410
Active Employees: 3699
Attrition Rate: 16.12%
R&D has the largest workforce.
Majority of employees belong to the 26–35 age group.
Average Salary is highest in the R&D department.
